# macros-for-coreldr
Привет!
Это первый мой макрос который я решил показать тебе. Он помогает расчитать цену изделия, а также точную площадь с периметром объекта . 
Метод установки макроса:
1. Скачай архив и перенеси в C:\Users\Твой профиль\AppData\Roaming\Corel\Версия корела\Draw\GMS
2. Распакуй и можешь закрыть проводник
3. Теперь переходи в Coreldraw
4. Открывай Инструменты -> Параметры -> Настройка -> Команды
5. Теперь выбери в выпадающем списке макросы
6. Чуть ниже найди "StartPanel.StartPanel" - это и будет ярлыком на макрос
7. Возьми его и перетащи куда будет удобно или можешь установить горячую клавишу
8. Пользуйся!.
Будут вопросы пиши отвечу.
